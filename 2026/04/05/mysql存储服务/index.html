MySQL 数据库基础管理总结（精简版）

1. 密码管理

操作 命令（5.7/8.0 通用示例） 说明
首次设置密码 mysqladmin password '新密码' 默认给 root 用户设置
修改密码 mysqladmin -uroot -p旧密码 password '新密码' 命令行方式
ALTER USER 'root'@'localhost' IDENTIFIED BY '新密码'; SQL 方式（5.7+）
UPDATE mysql.user SET authentication_string=PASSWORD('新密码') WHERE user='root'; 5.6 及更早
忘记密码重置 1. 停止 MySQL
2. mysqld_safe --skip-grant-tables --skip-networking &
3. 免密登录后 FLUSH PRIVILEGES;
4. 再执行 ALTER USER 改密码
5. 重启正常服务		 进入安全模式绕过授权表

2. 密码加密插件

  • 旧版默认：mysql_native_password（兼容性好）
  • 新版默认（8.0）：caching_sha2_password（可能导致旧客户端/主从连接失败）
  • 解决方案：
    • 创建/修改用户时指定：CREATE USER ... IDENTIFIED WITH mysql_native_password BY '密码'
    • 全局修改 /etc/my.cnf：[mysqld] default_authentication_plugin=mysql_native_password
