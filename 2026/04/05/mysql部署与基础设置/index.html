5. 报错

排错

  1. 看 MySQL 错误日志（核心）

  2. 检查服务状态

  3. 检查端口与权限

常见错误速查

现象 可能原因 排查方法
启动失败，ERROR [MY-010257] 端口 3306 被占用 netstat -tulnp | grep 3306，停用冲突进程或修改 port
Access denied for user 密码错误或认证插件不匹配 核对密码；MySQL 8.0 默认 caching_sha2_password，老客户端需改用 mysql_native_password[reference:3]
远程连接失败 bind-address 未放开 / 防火墙 / SELinux bind-address = 0.0.0.0；firewall-cmd --add-port=3306/tcp；setsebool -P mysqld_connect_any 1[reference:4]
temporary password 未生成 数据目录权限问题 chown -R mysql:mysql /var/lib/mysql
密码过期无法登录 首次改密后未设置 PASSWORD EXPIRE NEVER 临时以 skip-grant-tables 启动修复
Too many connections max_connections 太小 修改配置文件调大该值并重启
