NFS

安装—配置—启动—创建挂载目录

1安装nsf

yum -y install nfs-utils

2.配置nfs配置

vim /etc/exports #默认是没有的

/data 172.16.1.0/24(rw,sync,all_squash,anonuid=666,anongid=666) #默认是nobody65535

3.创建必须的配置文件

mkdir /data

chown nobody,nobody

4,启动nfs

systemctl start nfs

cat /var/lib/nfs/etab #里面有内容说明正确

5.客户端挂载

1.查看nfs客户端共享那个目录

showmount -e 172.16.1.31

2.挂载

mount -t nfs 172.16.1.31:/data /mnt

3.开机自动挂载

/etc/fatab

4.取消挂载

umount /mnt

5.查看

df -h

Cat /proc/mounts
